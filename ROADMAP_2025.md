# 🚀 BahaaIT Network Tools - Roadmap 2025

Ce document liste les fonctionnalités à venir pour transformer l'application en une suite professionnelle de niveau entreprise.

## 🔴 Haute Priorité (Système d'Alerte & Mobilité)
- [ ] **Notifications Webhooks** : Intégration Discord, Telegram et Slack pour les alertes DOWN.
- [ ] **Alertes Email (SMTP)** : Envoi automatique de rapports d'erreurs par courriel.
- [ ] **Application Mobile compagnon** : Visualisation du statut des serveurs sur smartphone.

## 🟠 Priorité Moyenne (Automatisation & Rapports)
- [ ] **Auto-Discovery (Scan ARP/ICMP)** : Scan automatique du réseau local pour détecter et ajouter les équipements.
- [ ] **Générateur de Rapports PDF** : Exportation de rapports d'uptime hebdomadaires/mensuels pour les clients.
- [ ] **Wake-on-LAN (WoL)** : Fonctionnalité pour réveiller les serveurs et PC à distance.

## 🔵 Priorité Basse (Outils Ingénierie & Design)
- [ ] **Calculateur de Subnet Pro** : Outil complet pour le calcul des masques et plages IP.
- [ ] **Topologie Réseau Graphique** : Visualisation sous forme de nœuds interactifs des connexions réseau.
- [ ] **Analyseur de Paquets (Mini-Wireshark)** : Capture basique de paquets pour le debug rapide.

## ✅ Terminé (Avril-Mai 2026)
- [x] Interface **Quantum Midnight 2025** (Mode Sombre Premium).
- [x] Monitoring IP avec historique **SQLite** (30 jours+) et export CSV.
- [x] Tableau de bord mondial (Apple, Netflix, AWS, etc.).
- [x] Gestionnaire de Sites et Favoris avec persistance.
- [x] Navigateur Web avec **toggle de sécurité SSL**.
- [x] Système d'alarme sonore personnalisable.
- [x] **Mode Ghost** (Réduction en zone de notification).
- [x] **Graphique de bande passante** temps réel.
- [x] **Scan ARP (Scapy)** avec adresses MAC.
